E-commerce Website

Overview:

This is a basic e-commerce platform built with HTML, CSS, JavaScript, PHP, and MySQL. It allows users to browse products, add them to the shopping cart, and proceed to checkout.

Features:

  1. Product Listings: Products are dynamically fetched from a MySQL database and displayed on the website.
  2. Add to Cart:Users can add products to their cart, which is stored in a session.
  3. View Cart: The cart page shows all added products with their details and total price.
  4. Database: MySQL database stores products, users, and orders.
  5. products table: Holds product details (name, price, description, etc.)
  6. users table: Stores user credentials (for optional registration/login)
  7. orders table: Stores order information (user, total price, etc.)

Technology Stack:

  1. Frontend: HTML, CSS (for design), JavaScript (optional for dynamic features).
  2. Backend: PHP (for processing actions like adding to cart and interacting with the database).
  3. Database: MySQL (for storing products, users, and orders).
